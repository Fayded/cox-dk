Creative Specs

on-air

What content is accepted?

Both SD and HD content is accepted in :10, :15, :30, :60 second spot lengths, as well as all long form commercial content (all content :120 seconds or longer).

What file formats are accepted?

Cox Media currently accepts video resolution of 720x480, 1280x720, or 1920x1080. Other resolutions will be rejected. The audio bit depth needs to be 16 bits. The maximum file size is 2GB.

MOV (Quicktime) file container using the H.264 codec is preferred.

in-banner video

Format

  • Audio Video Interleave (.avi)
  • Apple QuickTime Movie (.mov)
  • MPEG Video (.mpg or Mpeg)
  • MPEG-4 Video (.mp4)
  • Windows Media Video (.wmv)
  • Flash Video File (.flv)

Details

Upload the highest quality file you have. From a quality perspective, you’ll get the best results if the video meets at least these requirements:

  • Frames per second: 20
  • Resolution: 640x480 pixels
  • Bit rate: 1 megabit per second
  • Length: No longer than about 30 seconds in length.
  • File size: No larger than 500MB.

Note

On the Ad Network, there is limited inventory available for 15 second placements. 30 second placements offer more inventory. It is highly suggested to provide both 15 and 30 second spots to have the optimal number of sites for pre-roll video. File Size: no larger than 2.2 MB

Encoded formats
